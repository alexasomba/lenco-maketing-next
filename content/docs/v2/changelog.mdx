## Migration Guide

### Migrating from v1 to v2

<Steps>
<Step>
### Update Base URL

Change your API base URL:

```diff
- https://api.lenco.co/access/v1
+ https://api.lenco.co/access/v2
```
</Step>

<Step>
### Update Transfer Endpoints

v2 uses specific endpoints for each transfer type:

```javascript
// v1: Single endpoint for all transfers
POST /transactions
{
  "accountId": "...",
  "accountNumber": "0123456789",
  "bankId": "..."
}

// v2: Type-specific endpoints
POST /transfers/bank-account
POST /transfers/mobile-money
POST /transfers/lenco-money
POST /transfers/lenco-merchant
```
</Step>

<Step>
### Add Collections (Optional)

If accepting payments, integrate the new Collections API:

```javascript
// Mobile money collection
POST /collections/mobile-money
{
  "amount": 1000,
  "reference": "order-123",
  "phone": "0971234567",
  "operator": "airtel",
  "country": "zm"
}
```
</Step>

<Step>
### Update Webhook Handlers

New webhook events in v2:

| Event | Description |
|-------|-------------|
| `collection.successful` | Payment received |
| `collection.failed` | Payment failed |
| `collection.settled` | Funds settled to account |

</Step>
</Steps>
